Hugo创建文档网站

2020-11-07
1 min read

准备工作

  • 首先要有一个github账号
  • 下载Hugo：https://github.com/gohugoio/hugo/releases
  • 下载Git： https://git-scm.com/download/win

1、Hugo安装

在Hugo下载地址中根据不同操作系统，选择对应的压缩包。此处以Windows系统为例，如图：

安装适配

下载后解压，然后将hugo.exe所在的完整路径添加到环境变量中。

验证Hugo是否安装成功

不管在哪种系统中安装Hugo，最后我们都可以在命令行窗口使用下面命令查看Hugo是否安装成功：

hugo version

2、Git安装

进入git官网安装，选择Windows版本：

Git是否安装成功

具体方法

1、创建站点

Hugo和Git安装成功以后，在Git窗口输入命令创建站点：

hugo new site homework

新的站点文件夹homework将自动生成到当前目录下： 执行完 hugo new site homework命令后会得到一个包含以下文件的目录。

archetypes
config.toml  #配置文件；
content      #存储网站所有文档内容；
data         #数据；
layouts      #全局样式，优先级高于主题下的 layouts 文件夹；
static       #静态文件，优先级高于主题下的 static 文件夹；
themes       #主题目录;

2、下载主题

在Hugo主题列表中选择自己喜欢的主题进行下载，以Eureka为例：

点击Homepage，进入主题详情页面，可根据入门说明快速搭建一个Eureka主题的Hugo网站；

也可进入主题页面，点击Download，下载主题zip压缩包；

将下载解压好的文件复制到themes文件夹下，注意：此处需将文件命名为主题名字。

3、适配主题启动站点

根据Eureka主题说明，将主题文件夹中exampleSite中的config文件夹复制到项目根目录下，对config文件夹的内容进行配置。配置后删除原先的config.toml文件即可。

完成以上步骤后使用在Git窗口使用 hugo server命令，并在浏览器中输入地址http://localhost:1313/ 即可在预览项目：

hugo server

4、创建页面和文章

新建一个about页面:

hugo new about.md

about.md 自动生成到了 content/about.md，内容如下：

---
title: "About"
date: 2020-11-06T18:29:08+08:00
draft: true
---

文档编辑

使用Git对文档进行编辑

vim content/about.md

使用Visual Studio Code进行文档编写，需提前在左侧扩展中下载Markdown ALL in one和markdownlint插件；

其次，根据自身文章内容进行编辑，并预览；

创建第一篇文章，放到posts目录：

hugo new posts/first.md

first.md 自动生成到content/posts/first.md, 内容如下：

---
title: "First"
date: 2020-11-07T12:39:02+08:00
draft: true
---

文档编辑方法与上述方法相同，注意若想要将文章在站点显示需将draft:true 改为draft:false。

5、修改配置文件

站点Title修改：编辑器打开config.toml进行修改和保存，此处将title修改为作业；

站点菜单栏的修改：编辑器打开menus.toml进行修改和保存，此处修改菜单名称，并将多余菜单Docs进行注释。

6、将网站部署到GitHub Page上

打开GitHub，创建一个仓库，仓库名填写“用户名+github.io”，点击创建

在站点根目录执行 Hugo 命令生成最终页面：

hugo --theme=eureka --baseUrl="https://JasmineTao-zi.github.io/" --buildDrafts

部署完成后，所有静态页面都会生成到 public 目录

进入public文件目录下，对public文件夹进行初始化

git init

添加public中的所有文件：

git add .

提交(其中引号内为备注)：

git commit -m "Homework"

链接刚创建的GitHub仓库地址：

git remote add origin https://github.com/JasmineTao-zi/JasmineTao-zi.github.io.git

将pubilc目录里所有文件 push 到刚创建的Repository的 master 分支。

git push -u origin master
