Hugo创建文档网站
准备工作 首先要有一个github账号 下载Hugo：https://github.com/gohugoio/hugo/releases 下载Git： https://git-scm.com/download/win 1、Hugo安装 在Hugo下载地址中根据不同操作系统，选择对应的压缩包。此处以Windows系统为例，如图： 安装适配 下载后解压，然后将hugo.exe所在的完整路径添加到环境变量中。 验证Hugo是否安装成功 不管在哪种系统中安装Hugo，最后我们都可以在命令行窗口使用下面命令查看Hugo是否安装成功： hugo version 2、Git安装 进入git官网安装，选择Windows版本： Git是否安装成功 具体方法 1、创建站点 Hugo和Git安装成功以后，在Git窗口输入命令创建站点： hugo new site homework 新的站点文件夹homework将自动生成到当前目录下： 执行完 hugo new site homework命令后会得到一个包含以下文件的目录。 archetypes config.toml #配置文件； content #存储网站所有文档内容； data #数据； layouts #全局样式，优先级高于主题下的 layouts 文件夹； static #静态文件，优先级高于主题下的 static 文件夹； themes #主题目录; 2、下载主题 在Hugo主题列表中选择自己喜欢的主题进行下载，以Eureka为例： 点击Homepage，进入主题详情页面，可根据入门说明快速搭建一个Eureka主题的Hugo网站； 也可进入主题页面，点击Download，下载主题zip压缩包； 将下载解压好的文件复制到themes文件夹下，注意：此处需将文件命名为主题名字。 3、适配主题启动站点 根据Eureka主题说明，将主题文件夹中exampleSite中的config文件夹复制到项目根目录下，对config文件夹的内容进行配置。配置后删除原先的config.toml文件即可。 完成以上步骤后使用在Git窗口使用 hugo server命令，并在浏览器中输入地址http://localhost:1313/ 即可在预览项目： hugo server 4、创建页面和文章 新建一个about页面: hugo new about.md about.md 自动生成到了 content/about.md，内容如下： --- title: &quot;About&quot; date: 2020-11-06T18:29:08+08:00 draft: true --- 文档编辑 使用Git对文档进行编辑
